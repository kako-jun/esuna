# Esuna — 概要

## 名前の由来

「エスナ」はファイナルファンタジーの回復魔法。状態異常を癒す魔法の名前を冠したこのアプリは、視覚障害という「状態異常」を抱えながらも、日常の趣味を諦めずに生きることを支援するために作られた。

## 動機

失明が怖い。それは多くの人が心のどこかに持つ恐怖だ。

失明した後、これまで楽しんでいた趣味を手放さなければならないとしたら、どれほど絶望するだろう。ニュースを読む、SNSで人と繋がる、物語に浸る。それらが突然できなくなる未来を想像すると、胸が痛い。

アプリ開発者として、その絶望に先回りして準備しておくことこそが仕事だと思った。失明してから途方に暮れるのではなく、「Esunaがあるから大丈夫」と言える世界を作りたい。

## 解決したい問題

スマートフォンには物理ボタンがない。どこをタップすれば何が起きるか、見えなければわからない。パソコンのマウスオーバーに相当する情報提示の仕組みが、タッチ操作には本質的に欠けている。

既存の視覚障害者向けアプリは多くがWindows 95時代の設計を引きずっており、現代のWebサービスには対応できていない。OSに付属する読み上げ機能は汎用的すぎて、アプリ固有の文脈を伝えるには不十分だ。そもそも、アプリをインストールする作業自体が、補助なしには難しい。

## Esunaの答え

Webアプリとして提供することで、インストール不要・アップデート自動・どのOSでも同じように動くという利点を得る。

さらに、全画面表示で使い続けられるポータルとして設計する。ニュースも、SNSも、5chも、音楽も、ゲームも——外部サイトに遷移せず、Esunaの中で完結する。Esunaの操作ルールを一度覚えてしまえば、すべての機能を同じ感覚で使える。

## 9マスUIという統一操作体系

Esunaの中心にある設計思想は「9マスグリッド」だ。画面を3×3のマスに分割し、各マスに意味を持たせる。

- 左上ほど重要、右下ほど補助的
- 「戻る」は常に同じ位置
- 数字キー1〜9でどこからでも直接選択できる

ユーザーはタップ位置を正確に認識できなくてよい。画面のどのあたりを触ったかで操作が決まる。この体系をEsunaのすべての画面で貫くことで、「Esunaならわかる」という安心感が生まれる。

## 目指す世界

国を問わず、OSを問わず、年齢を問わず。視覚に障害がある人が、Esunaを開けば絶望しない世界。

それが、このプロジェクトの目指すゴールだ。
