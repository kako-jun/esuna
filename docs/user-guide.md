# Esuna — 使い方ガイド

## 起動する

ブラウザで Esuna の URL を開く。起動すると自動的に全画面表示になり、ホーム画面が表示される。

起動と同時に、現在地と操作方法の読み上げが始まる。「ホーム画面です。1から9のキーでエリアを選択できます」のようなアナウンスが流れる。

初めて使うときは、このアナウンスをそのまま聞いていれば操作を覚えられるよう設計されている。

## 9マスの操作

画面は9つのエリアに分かれている。

**タッチ操作の場合**
画面の左上あたりを軽く触ると「戻る」エリア。真ん中を触ると「メインコンテンツ」エリア。右端を触ると右列のエリア。タッチ位置が多少ずれても、最も近いエリアに反応する。

**エリアに触れたとき**
そのエリアの名前と現在の内容が読み上げられる。「メインコンテンツ。はてなブックマーク、人気エントリー。本日2位。タイトルは……」のように教えてくれる。

## テンキーでの選択

数字キー 1〜9 を押すと、対応するエリアが直接選択される。

```
7 8 9
4 5 6
1 2 3
```

テンキーの配置と画面のエリアが対応している。テンキーの「5」を押せば中央のメインコンテンツが選択される、という感覚で操作できる。

矢印キーでエリアを移動し、Enterで決定することもできる。

## 読み上げを操作する

- エリアを選択すると、内容の読み上げが始まる
- `停止` または Escape を押すと、その時点の読み上げキューを破棄して停止する
- 現在の実装では `停止` は `一時停止` ではない
- 停止した続きから自動で再開する機能は、まだない
- もう一度選択すると最初から読み上げ直す
- 設定エリア（3番）から読み上げ速度と音量を変更できる

長い記事を読み上げているとき、途中で止めてから「次へ」エリアで次の記事に進む、という流れが基本的な操作パターンだ。

### 停止の現在仕様

`停止` は確認付きの操作ではない。1回で即時実行される。

- 読み上げ中の本文を止める
- まだ再生されていない後続の読み上げ予約も消える
- `停止` のあとに自動で続きが再開されることはない

つまり現在の `停止` は、`一時停止 / 再開` ではなく `中断` に近い。

### 自動ナビゲーションとの関係

自動ナビゲーションを有効にしている場合でも、`停止` の意味は「その場で読み上げを止める」ことが優先されるべきである。

ただし、現在の実装ではこの仕様が十分に整理されていない。`停止` と `読み終わり` の区別、`停止後にどう扱うか` は今後の改善対象である。

## コンテンツを切り替える

ホーム画面の下列（7・8・9番エリア）にコンテンツの種類が並んでいる。

- 7番：はてなブックマーク
- 8番：5ch
- 9番：SNS

選択したコンテンツの画面に移り、一覧が読み上げられる。気になった記事を選ぶと詳細が開く。

「戻る」（1番エリア）でいつでも前の画面に戻れる。

## リロードする

コンテンツを最新の状態に更新したいときは、2番エリア（リロード）を選択する。読み込み中は「取得中です」とアナウンスされ、完了すると新しいコンテンツの件数が伝えられる。

## 設定を変える

3番エリア（設定）を選ぶと、設定画面が開く。

- 読み上げ速度（遅い・普通・速い）
- 読み上げ音量
- 読み上げの詳細度（簡潔・標準・詳細）
- 表示言語

変更は即座に反映され、次の読み上げから適用される。
