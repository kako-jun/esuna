# Esuna 今後の実装予定機能

視覚障害者の方が毎日を前向きに生きるための機能を優先順位順に記載

## 🌟 最優先：毎日の習慣になる機能

### 1. 時刻読み上げ・天気予報（毎朝の日課）
**実装難易度**: ⭐☆☆☆☆ (簡単)
**優先度**: 最高

```
起動時に「おはようございます。今日は○月○日、午前7時30分です。
東京の天気は晴れ、最高気温25度、最低気温18度です」
```

- **理由**：毎日必要な情報。これがあるとアプリを開く習慣ができる
- 外出の判断ができる = 自立につながる
- **実装方法**：
  - OpenWeather API（無料枠で十分）
  - 起動時に自動読み上げ
  - 設定で地域を選択可能

**実装状況**: ✅ 実装済み

### 2. RSSニュースリーダーUI
**実装難易度**: ⭐☆☆☆☆ (簡単)
**優先度**: 高

- **理由**：ライブラリ（rss.ts）は既に実装済み。UIコンポーネントを追加するだけ
- 自分で好きなニュースサイトを選べる
- はてブとは違う情報源
- **実装方法**：
  - 既存のrss.tsを使用
  - FivechBoardListと同様のUI構造
  - デフォルトフィード一覧から選択
  - 9グリッドナビゲーションで操作

**実装状況**: ✅ 実装済み

### 3. ラジオ（radiko、NHKらじるらじる）
**実装難易度**: ⭐⭐⭐☆☆ (中程度)
**優先度**: 高

- **理由**：人の声が常に聞こえる環境は孤独を和らげる
- リアルタイムの社会とつながっている実感
- 「ながら聞き」で一日中楽しめる
- **実装方法**：
  - radiko: エリア判定が必要だが、ストリーミングURL取得は可能
  - NHKらじるらじる: 比較的シンプル
  - audio要素で再生
  - 番組表取得も検討

**実装状況**: 🔜 未実装

### 4. 音声メモ・日記
**実装難易度**: ⭐⭐☆☆☆ (やや簡単)
**優先度**: 高

- **理由**：思ったことをすぐ記録できる = 記憶の不安解消
- 感情を言語化することで心の整理
- 「今日あった良いこと」を録音して聞き返せる = 前向きな気持ち維持
- **実装方法**：
  - MediaRecorder API

  - 日付・タグで検索

## 技術的な検討事項

### ブラウザ制約
- **ファイル選択不可**: 見えないので使えない → 予め登録されたコンテンツリストから選択
- **通知**: Web Notifications APIは許可が必要だが音声案内でカバー
- **バックグラウンド再生**: PWA化すれば可能

### API制約
- **radiko**: エリア判定、暗号化あり。ライブラリ活用で実装可能
- **YouTube**: YouTube IFrame Player API使用可能だがログイン必要
- **Spotify**: Web Playback SDK使用可能だがプレミアム会員必要

### アクセシビリティ
- すべての機能を9グリッドナビゲーションで操作可能に
- 音声ガイダンスを常に提供
- キーボード操作完全対応
- 自動ナビゲーション機能の拡充
